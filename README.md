# MeetHub - Seamless Meeting Collaboration for Teams

![image](https://github.com/user-attachments/assets/265228ab-da63-4a24-92b0-e53b06d48350)

## Overview

MeetHub is a modern meeting management application built with Next.js, Tailwind CSS, and shadcn/ui components. It provides teams with a seamless platform to schedule, manage, and conduct virtual meetings efficiently.

## Features

- **User Authentication**: Secure sign-up and sign-in functionality
- **Meeting Scheduling**: Easy-to-use interface for scheduling meetings
- **Video Conferencing**: High-quality video calls with multiple participants
- **Meeting Management**: Create, edit, and delete meetings
- **Participant Management**: Invite and manage meeting participants
- **Responsive Design**: Works seamlessly across desktop and mobile devices

## Tech Stack

- **Frontend Framework**: [Next.js](https://nextjs.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **UI Components**: [shadcn/ui](https://ui.shadcn.com/)
- **Deployment**: Vercel
